MK11 Korean Patch: CLI Installer
by KimHerV
================================================================================

MK11 한글 패치 CLI 설치 스크립트입니다.
Windows 11/10/8/7, 백신에 차단된 PC, Steam Deck, Linux (Bazzite 포함)
등 다양한 환경에서 동작합니다.

GitHub Releases에서 최신 한글 패치를 받아 게임 폴더에 적용합니다.

1. 설치 안내

================================================================================

1.1. 설치되는 항목

* 한글 번역 텍스트 (53,000+ 항목)
* 한글 글리프 텍스처 (나눔스퀘어 네오)
* 번역 로더 플러그인 (선택, 설치 [3/6] 단계에서 결정)

원본 파일은 설치 전 자동 백업되며, 게임 폴더의 _backup_korean_patch/
에 보존됩니다.

--------------------------------------------------------------------------------

1.2. Windows 설치

1. install.bat을 더블 클릭합니다.
2. PowerShell 창이 열리고 단계별 진행이 표시됩니다.
3. 완료 후 Steam에서 MK11 언어를 "중국어 간체"로 변경합니다.

install.bat은 단순 런처입니다. 실제 로직은 install.ps1에 있으며,
한글 텍스트 인코딩 안정성을 위해 PowerShell에 위임합니다.

--------------------------------------------------------------------------------

1.3. Steam Deck / Linux (Bazzite 포함) 설치

데스크탑 모드 터미널에서 다음을 실행합니다:

    bash install.sh

또는 실행 권한을 부여한 뒤 직접 실행합니다:

    chmod +x install.sh
    ./install.sh

완료 후:

1. Steam -> MK11 속성 -> 언어 -> 중국어 간체
2. 게임을 실행하여 한글 표시 여부를 확인합니다.
3. 한글이 표시되지 않으면 Steam 시작 옵션에 다음을 추가합니다:

    WINEDLLOVERRIDES="dinput8=n,b" %command%

4. 그래도 표시되지 않으면 사용 중인 Proton 버전과 함께 GitHub
   Issues에 제보 부탁드립니다.

2. 번역 로더 플러그인 (CVD 우회)

================================================================================

2.1. 무엇인가

MK11 2026 빌드는 게임 파일 무결성을 검증하며, 수정된 파일
(한글 패치 포함)을 로딩하지 않을 수 있습니다.

번역 로더 플러그인은 메모리에서 무결성 검사를 비활성화하여
한글 패치가 정상 동작하도록 합니다. 디스크 파일은 변경되지
않습니다.

설치 시 [3/6] 단계에서 적용 여부를 결정할 수 있습니다.

--------------------------------------------------------------------------------

2.2. 설치되는 파일

번역 로더 플러그인을 설치하기로 선택한 경우, 게임 폴더의
Binaries/Retail/ 에 다음 5개의 오픈소스 파일이 추가됩니다.

* dinput8.dll                Ultimate ASI Loader, MIT
* ASIMK11.asi, ASIMK11.ini   CVD 우회 본체
* libzmq, libsodium          런타임 의존성

--------------------------------------------------------------------------------

2.3. 본 플러그인이 수행하지 않는 일

* 치트, 언락, DLC 우회, 승부 조작 등 부정 행위
* 게임 파일 디스크 변경 (메모리 패치만 수행)
* 게임 폴더 외 다른 파일, 프로그램 변경
* 외부 서버와의 통신 또는 사용자 정보 수집

3. 업데이트

================================================================================

3.1. 새 버전 적용 방법

GitHub Releases에 새 버전이 올라오면 install.bat 또는 install.sh를
다시 실행하기만 하면 됩니다.

설치 흐름은 동일합니다. 자동 감지된 게임 폴더를 확인하고, [3/6]
단계에서 번역 로더 플러그인 설치 여부를 결정한 뒤 진행하면 게임
폴더 파일이 최신 버전으로 교체됩니다.

--------------------------------------------------------------------------------

3.2. 백업 동작

* 백업 폴더 _backup_korean_patch/ 에는 첫 설치 시점의 원본 파일이
  보존됩니다.
* 재실행 시 기존 백업이 발견되면 덮어쓰지 않습니다.
* 이 정책으로 언제든 첫 설치 시점의 원본으로 정확히 복구할 수
  있습니다.

4. 제거

================================================================================

4.1. uninstall 스크립트 사용

Windows

  uninstall.bat 더블 클릭

Steam Deck / Linux

  데스크탑 모드 터미널에서:

    bash uninstall.sh

스크립트가 다음을 자동으로 처리합니다.

* _backup_korean_patch/ 의 백업 파일을 게임 폴더로 복원
* 번역 로더 플러그인이 설치되어 있으면 Binaries/Retail/ 에서
  5개 ASI 파일 제거

복원 후 Steam -> MK11 속성 -> 언어 설정에서 원하는 언어로 다시
변경하면 패치 적용 전 상태로 돌아갑니다.

백업 폴더는 보존됩니다. 필요 없으시면 직접 삭제해 주세요.

--------------------------------------------------------------------------------

4.2. 수동 복구

스크립트를 사용하기 어려운 경우, 다음 위치에서 원본 파일을 직접
복사해 되돌릴 수 있습니다.

    _backup_korean_patch/Coalesced.CHS              -> Localization/
    _backup_korean_patch/ui_c_inGameFonts_chs.xxx   -> Asset/

번역 로더 플러그인을 제거하려면 게임 폴더의 Binaries/Retail/ 에서
다음 5개 파일을 삭제하면 됩니다.

    dinput8.dll
    ASIMK11.asi
    ASIMK11.ini
    libsodium.dll
    libzmq-v120-mt-4_3_4.dll

5. 코드 공개 및 라이센스

================================================================================

5.1. 수행 작업 / 수행하지 않는 작업

이 스크립트는 의도적으로 단순하고 짧게 작성되었습니다. 코드
전체가 공개되어 있으므로 누구나 동작을 확인할 수 있습니다.

수행 작업:

* Steam 라이브러리 자동 탐색 (레지스트리 + libraryfolders.vdf 파싱)
* GitHub Releases API 호출 (최신 버전 확인)
* 패치 파일 다운로드 (콘텐츠 2개, 선택 시 CVD 우회 zip)
* SHA256 무결성 검증 (번역 로더 플러그인 적용 시)
* 원본 백업 후 게임 폴더에 적용

수행하지 않는 작업:

* 외부 서버 통신 (GitHub Releases 제외)
* 사용자 정보 수집
* 게임 폴더 외 시스템 변경
* 레지스트리 변경
* 백그라운드 프로세스
